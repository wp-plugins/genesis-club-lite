FAQ Accordion, User Signatures, Responsive Menus, Responsive Logos, Label overrides and lots more for Genesis Child Themes.

== Description ==
* Include logo image in the title instead of text and tagline
* Add an author signature at the foot of each post
* Add extra widget areas at the top or foot of each post or archive page
* Change "read more..." label 
* Change commentary label "Speak Your Mind"
* Change breadcrumb labels such "Archives for"
* Add a responsive hamburger menu icon to your navigation bars
* Add the accordion feature to your FAQ page or FAQ category
* Facebox LikeBox widget
* Enables Simple Social Icons plugin to have multiple widgets on the same page with different sizes
* Exclude selected pages from the Genesis 404 page, which is a sitemap of all pages on the site - typically you will want to exclude landing pages, "thank you" pages and any download pages.
* Compatible with both Genesis 1.9.x and Genesis 2.x with HTML5 support

== Installation ==
1. Use the standard WordPress plugin automatic updates system for updating to the latest version or use the manual steps below. 
1. Uncompress the downloaded zip archive in [WordPress install root]/wp-content/plugins
1. Activate the plugin in your WordPress plugins control panel
